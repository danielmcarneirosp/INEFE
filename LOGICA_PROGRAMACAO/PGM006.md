# PGM006 - GRAVAR UM REGISTRO NO ARQUIVO

1. Definir Registro com Alfanumércio de 50 bytes.
2. Definir NOME com Alfanumérico de 40 bytes
3. Definie DATA-NASCIMENTO com Alfanumérico de 10 Bytes.
4. Pedir o Usuário o Nome
5. Pedir a Data de Nascimento.
6. MOVER o primeiro ACCEPT digitado para a Variável NOME
7. MOVER o segundo ACCEPT digitado para a Variável DATA-NASCIMENTO
8. Abrir o Arquivo.
9. Verificar se o Arquivo existe.
10. Senão existir Criar o Arquivo.
11. Escrever no Arquivo
12. Se for tudo OK, imprimir mensagem que o resitro foi gravado com sucesso
13. Senão: Imprimir uma mensagem de erro.
14. Fechar o Arquivo.
15. Final de Processamento.
