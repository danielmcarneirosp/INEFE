# PGM007 - GRAVAR N REGISTRO NO ARQUIVO

1. Definir Registro com Alfanumércio de 50 bytes.
2. Definir NOME com Alfanumérico de 40 bytes
3. Definie DATA-NASCIMENTO com Alfanumérico de 10 Bytes.
4. Pedir a Entrada do Nome e Data Nascimento até o Usuário responder o passo 16.
5. Pedir o Nome.
6. Pedir a Data de Nascimento.
7. MOVER o primeiro ACCEPT digitado para a Variável NOME
8. MOVER o segundo ACCEPT digitado para a Variável DATA-NASCIMENTO
9. Abrir o Arquivo.
10. Verificar se o Arquivo existe.
11. Senão existir Criar o Arquivo.
12. Escrever no Arquivo
13. Se for tudo OK, imprimir mensagem que o resitro foi gravado com sucesso.
14. Senão: Imprimir uma mensagem de erro.
15. Fechar o Arquivo.
16. Perguntar o Usuário se deseja Cadastrar mais um Registro?
17. Se Sim, Ir para o passo 4.
18. Senão, Ir para o Passo 18.
19. Final de Processamento.
