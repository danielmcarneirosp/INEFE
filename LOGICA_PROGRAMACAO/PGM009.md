# PGM009 - LER UM ARQUIVO DE ENTRADA E GRAVAR UM REGISTRO DE SAIDA

1. Arquivo de Processamento com REGISTRO DE 80 bytes.
2. Layout: NOME como Alfanumérico de 40 bytes
   EMAIL como Alfanumérico de 40 bytes.
3. Abrir o Arquivo para Leitura
4. Abriu com Sucesso?
5. Se Sim
6. Ler o Registro de Entrada.
7. Abrir o Arquivo de Saída.
8. Se tudo OK?
9. MOVER o conteúdo do NOME PARA o NOME do Arquivo de Saída.
10. MOVER o conteúdo do EMAIL PARA o EMAIL do Arquivo de Saída.
11. Escrever no Arquivo de Saída
12. Fechar o Arquivo de Saída.
13. Movimentar para o segundo Registro do Arquivo de Leitura
14. É final de Arquivo?
15. Senão?
16. Ir para o Passo 7.
17. Se Sim.
18. Fechar o Arquivo de Entrada.
19. Final de Processamento.
20. Senão Passo 4.
21. Imprimir Mensagem de Erro informando que não conseguiu abrir o Arquivo de Entrada.
22. Ir para o Passo 19.
