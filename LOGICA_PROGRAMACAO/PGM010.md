# PGM009 - LER UM ARQUIVO DE ENTRADA E GRAVAR UM REGISTRO DE SAIDA EM DOIS ARQUIVOS.

1. Arquivo de Processamento com REGISTRO DE 80 bytes.
2. Layout: NOME como Alfanumérico de 40 bytes
   EMAIL como Alfanumérico de 40 bytes.
3. Abrir o Arquivo para Leitura.
4. Abriu com Sucesso?
5. Se Sim
6. Ler o Registro de Entrada.
7. Abrir o Arquivo01 de Saída.
8. Abrir o Arquivo02 de Saída.
9. Se tudo OK?
10. MOVER o conteúdo do NOME PARA o NOME do Arquivo01 de Saída.
11. MOVER o conteúdo do EMAIL PARA o EMAIL do Arquivo01 de Saída.
12. Escrever no Arquivo de Saída01
13. Fechar o Arquivo de Saída01.
14. MOVER o conteúdo do NOME PARA o NOME do Arquivo02 de Saída.
15. MOVER o conteúdo do EMAIL PARA o EMAIL do Arquivo02 de Saída.
16. Escrever no Arquivo de Saída02
17. Fechar o Arquivo de Saída02.
18. Movimentar para o segundo Registro do Arquivo de Leitura
19. É final de Arquivo?
20. Senão?
21. Ir para o Passo 7.
22. Se Sim.
23. Fechar o Arquivo de Entrada.
24. Final de Processamento.
25. Senão abriu o Arquivo de Leitura no Passo 4.
26. Imprimir Mensagem de Erro informando que não conseguiu abrir o Arquivo de Entrada.
27. Ir para o Passo 24.
